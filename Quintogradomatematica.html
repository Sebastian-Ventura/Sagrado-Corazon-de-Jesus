Unidad 1: Derivadas Básicas

Teoría: La derivada es una operación matemática que mide el cambio instantáneo de una función respecto a su variable independiente. Su interpretación geométrica es la pendiente de la recta tangente a la gráfica en un punto específico.

Fórmula básica: f'(x) = lim (h→0) [f(x + h) - f(x)] / h

Las derivadas son fundamentales en física para estudiar velocidad, aceleración y en economía para optimizar costos y beneficios.

Reglas importantes:

  • Potencias: d/dx [xⁿ] = n·xⁿ⁻¹
  • Suma y resta: d/dx [f(x) ± g(x)] = f'(x) ± g'(x)
  • Constantes: d/dx [c] = 0

Ejemplos:

  • f(x) = x² → f'(x) = 2x
  • f(x) = 3x³ - 5x² + 2 → f'(x) = 9x² - 10x
  • f(x) = 7 → f'(x) = 0

Unidad 2: Límites de Funciones

Teoría: El límite es el valor al que se aproxima una función cuando su variable se acerca a un número específico. Permite analizar el comportamiento de las funciones cerca de ciertos puntos, incluso si la función no está definida allí.

Fórmula básica: lim(x→a) f(x)

Casos comunes:

  • Limites directos: Solo reemplaza el valor.
  • Indeterminaciones (0/0): Se resuelve factorizando o aplicando regla de L'Hôpital.

Ejemplos:

  • lim(x→2) (x² + 3) = 2² + 3 = 7
  • lim(x→3) (x² - 9)/(x - 3): Factorizamos → [(x - 3)(x + 3)] / (x - 3) → Límite: 6
  • lim(x→0) sin(x)/x = 1 (límite notable)

Unidad 3: Álgebra y Funciones Polinómicas

Teoría: Las funciones polinómicas son expresiones algebraicas con potencias enteras no negativas. Su gráfica es continua y suave. Se representan como f(x) = aₙxⁿ + aₙ₋₁xⁿ⁻¹ + … + a₀.

Propiedades:

  • Grado: Mayor exponente.
  • Raíces: Valores donde f(x) = 0.
  • Intersección: Cuando x = 0, f(0) es la intersección con el eje Y.

Ejemplos:

  • f(x) = 2x² - 3x + 1 → Polinomio de segundo grado.
  • f(x) = x³ - 6x² + 11x - 6 → Raíces: x = 1, 2, 3.
  • f(x) = -x⁴ + 4x² → Grado 4, con raíces en x = -2, 0, 2.

Unidad 4: Geometría del Espacio

Teoría: Estudia figuras tridimensionales y sus propiedades. Las principales son prismas, cilindros, pirámides, conos y esferas. Se analizan el volumen y las áreas.

Fórmulas básicas:

  • Volumen del prisma: Área de la base × altura.
  • Volumen del cilindro: πr²h.
  • Volumen de la pirámide: (Área de la base × altura) ÷ 3.
  • Volumen de la esfera: (4/3)πr³.

Ejemplos:

  • Prisma rectangular con base 4×3 y altura 5: Volumen = 4×3×5 = 60.
  • Cilindro con radio 3 y altura 7: Volumen ≈ 3.1416×9×7 ≈ 197.9.

Unidad 5: Estadística Inferencial (Introducción)

Teoría: La estadística inferencial permite extraer conclusiones sobre una población, usando datos de una muestra. Involucra estimaciones, pruebas de hipótesis y márgenes de error.

Conceptos clave:

  • Muestra: Subconjunto de la población.
  • Parámetro: Valor exacto de la población (desconocido).
  • Estadístico: Valor calculado de la muestra.
  • Error muestral: Diferencia entre el parámetro y el estadístico.

Ejemplos:

  • De 500 estudiantes, se encuesta a 50 sobre sus horas de estudio. El promedio muestral es una estimación del promedio poblacional.
  • Se estima con un 95% de confianza que el promedio de calificaciones está entre 14 y 16 puntos.
