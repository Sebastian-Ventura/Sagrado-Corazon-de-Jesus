Unidad 1: Ecuaciones Lineales

Resumen: Las ecuaciones lineales son aquellas que representan rectas y tienen la forma ax + b = 0. Su solución es un valor que hace verdadera la igualdad.

Ejemplo: Resolver 2x + 3 = 7 → 2x = 4 → x = 2.

Pregunta: ¿Cuál es la solución de 3x - 9 = 0?

Respuesta: x = 3.

Ejercicio: Resuelve las ecuaciones:
a) 5x + 2 = 12
b) 4x - 7 = 9.

Unidad 2: Sistema de Ecuaciones

Resumen: Un sistema de ecuaciones es un conjunto de ecuaciones que deben resolverse simultáneamente. Puede resolverse por el método de sustitución o igualación.

Ejemplo: Resuelve el sistema:
x + y = 5
x - y = 1
Solución: x = 3, y = 2.

Pregunta: ¿Cuál es la solución del sistema:
2x + y = 8
x + y = 5?

Respuesta: x = 3, y = 2.

Ejercicio: Resuelve:
a) x + 2y = 10
b) 3x - y = 5.

Unidad 3: Funciones

Resumen: Una función es una relación entre dos conjuntos, donde a cada valor del dominio le corresponde un único valor del rango. Se representan como f(x).

Ejemplo: f(x) = 2x + 1 → Si x = 2, entonces f(2) = 2(2) + 1 = 5.

Pregunta: ¿Cuál es el valor de f(3) si f(x) = 4x - 1?

Respuesta: f(3) = 4(3) - 1 = 11.

Ejercicio: Calcula f(0), f(2) y f(5) para f(x) = 3x + 2.

Unidad 4: Geometría - Triángulos y Polígonos

Resumen: La geometría estudia las figuras y sus propiedades. En esta unidad aprenderás sobre triángulos, polígonos y sus áreas y perímetros.

Ejemplo: El perímetro de un triángulo con lados 5 cm, 7 cm y 9 cm es 5 + 7 + 9 = 21 cm.

Pregunta: ¿Cuál es el área de un triángulo de base 6 cm y altura 4 cm?

Respuesta: Área = (6 × 4) ÷ 2 = 12 cm².

Ejercicio: Calcula el perímetro de un hexágono regular si cada lado mide 4 cm.

Unidad 5: Estadística Básica

Resumen: La estadística analiza datos para obtener conclusiones. Aprenderás sobre promedio, moda y mediana.

Ejemplo: Dado el conjunto de datos: 3, 4, 4, 5, 6, la media es (3 + 4 + 4 + 5 + 6) ÷ 5 = 4.4.

Pregunta: ¿Cuál es la moda del conjunto: 2, 3, 3, 5, 6?

Respuesta: Moda = 3.

Ejercicio: Calcula la media y mediana del conjunto: 1, 2, 2, 3, 4, 5.
